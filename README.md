# 💰 お小遣い＆バイト代管理帳 (グラフ付き)

PythonのWebフレームワークである **Streamlit** を使用して作成する、シンプルで楽しいお小遣い管理（家計簿）アプリです。
貯金目標を設定し、ゲーム感覚で目標達成率の進捗バーを伸ばしながら、自分の収支（収入と支出）をグラフで視覚的に管理できます。

---

## 🚀 主な機能と仕様

### 1. 貯金目標の設定（サイドバー）
- 目標金額（貯金目標）を入力・設定できます。
- **目標達成メーター（プログレスバー）**: 「現在の残高 / 貯金目標」の比率を計算し、目標達成率を % で表示するとともに、進捗バー（`st.progress`）を表示します。

### 2. 今月のサマリー（メイン画面上部）
- **現在の残高**: これまでのすべての「収入 - 支出」の合計額を表示します。
- **今月の使える残り金額**: 今月使える目標（任意）に対して、あといくら残っているかを表示します。

### 3. 収支データの入力（メイン画面中部）
- 以下のフォームから収支を入力できます。
  - **日付**: 日付選択（`st.date_input`）
  - **タイプ**: 「収入」または「支出」の選択（`st.radio`）
  - **カテゴリ**: 
    - 収入カテゴリ: 「お小遣い」「バイト代」「お年玉・お祝い」「その他」
    - 支出カテゴリ: 「買い食い」「ゲーム・ホビー」「洋服・美容」「友達との遊び」「勉強・本」「その他」
  - **金額**: 金額入力（`st.number_input`）
  - **メモ**: 自由記述（`st.text_input`）

### 4. 履歴の確認と削除（メイン画面下部）
- 入力されたすべてのデータがテーブル形式（`st.dataframe`）で一覧表示されます。
- 各行の横に「削除」ボタンがあり、間違えて入力したデータを削除できます。

### 5. グラフによる収支の視覚化（メイン画面最下部）
- **カテゴリ別支出グラフ**: pandasを用いて `type` が「支出」のデータのみを `category` ごとに集計し、棒グラフ（`st.bar_chart`）で表示します。
- **日々の累積残高グラフ**: 日付順に並べた収支データから、収入（+）・支出（-）を反映した累積残高を計算し、折れ線グラフ（`st.line_chart`）で表示します。
- データが1件もない場合は、グラフの代わりに案内メッセージを表示します。

## 💾 データ構造 (allowance_data.json)

データはローカルの `allowance_data.json` に保存されます。スキーマ（構造）は以下の通りです。

    {
        "goal": 10000,
        "transactions": [
            {
                "id": "1",
                "date": "2026-06-01",
                "type": "収入",
                "category": "お小遣い",
                "amount": 5000,
                "memo": "6月のお小遣い"
            },
            {
                "id": "2",
                "date": "2026-06-02",
                "type": "支出",
                "category": "買い食い",
                "amount": 500,
                "memo": "放課後のアイス"
            }
        ]
    }

## 🛠️ セットアップ手順

### 1. 仮想環境の準備と有効化

    python -m venv venv
    # Windowsの場合
    venv\Scripts\activate
    # macOS/Linuxの場合
    source venv/bin/activate

### 2. 必要なライブラリのインストール

    pip install -r requirements.txt

### 3. アプリの起動

    streamlit run app.py
